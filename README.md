# Prédiction du succès d’un produit e-commerce

## Description générale
Ce projet a pour objectif de prédire si un produit e-commerce sera populaire ou non à partir du dataset "Women's E-Commerce Clothing Reviews".  
Pour l’instant, ce README contient seulement une présentation générale. Les détails sur le prétraitement, les modèles et les métriques seront ajoutés plus tard.

## Dataset
- Source : [Kaggle - Women's E-Commerce Clothing Reviews](https://www.kaggle.com/datasets/nicapotato/womens-ecommerce-clothing-reviews)
- Contenu : avis clients, notes, recommandation, catégories de produit.

## Structure du projet actuelle
- `data/` → contient le dataset CSV  
- `notebook/` → notebook unique (sera utilisé pour l’EDA, preprocessing et ML)  
- `app/` → dossier pour l’application Streamlit (vide pour l’instant)
